# js13k-3dphysics-for-xem

Install:

```bash
npm i
```

Run dev server:

```bash
npm run dev
```

By default, the dev server will be available at <http://localhost:5173/>.

Build:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```

By default, the preview server will be available at <http://localhost:4173/>.
